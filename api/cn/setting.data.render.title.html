Function(title, treeNode)setting.data.render.title

概述[ 依赖 jquery.ztree.core 核心 js ]

用于渲染节点标题时进行重新渲染节点标题，最终节点标题将以方法返回的值为准

默认值：null

Function 参数说明

titleString

对应 zTree 利用 setting.key.title 获取到的节点标题

treeNodeJSON

待渲染节点的 JSON 数据对象

针对根进行渲染时，treeNode = null

返回值Boolean

返回值是 渲染后的节点标题

setting & function 举例


	var setting = {
		data: {
			render: function(title,treeNode){
				return title + treeNode.id;
			}
		}
	};
